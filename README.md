# Recipe Sharing Website 🍲

A recipe sharing website where users can create, share, and discover recipes.

## Features

- User registration and login
- Add recipes
- Edit recipes
- Delete recipes
- Search recipes
- View recipe details
- User profile
- Favorite recipes

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Firebase
- Git
- GitHub
